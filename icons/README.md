# Icons

이 디렉토리에는 크롬 익스텐션 아이콘 파일들이 위치합니다.

필요한 아이콘 파일들:
- `icon16.png` - 16x16 픽셀
- `icon48.png` - 48x48 픽셀
- `icon128.png` - 128x128 픽셀

아이콘을 직접 추가하시거나, 온라인 아이콘 생성기를 사용하실 수 있습니다.

임시로 다음 사이트에서 아이콘을 생성할 수 있습니다:
- https://www.favicon-generator.org/
- https://realfavicongenerator.net/

또는 간단한 색상 배경의 PNG 파일로 시작하셔도 됩니다.
